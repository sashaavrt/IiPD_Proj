Задача 4.
Записать логические выражения, которые имеют значение «Истина» только при выполнении указанных условий:
а) x < 2 и у < 3;
б) неверно, что x < 2;
в) x < 1 или y < 2;
г) неверно, что x < 0 и x < 5;
д) x < 0 и у > 5;
е) 10 < x < 20;
ж) x > 3 или x < 1;
з) 0 < y < 4 и x < 5;
и) х > 3 и x < 10.
