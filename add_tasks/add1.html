Сочетание операторов - проверьте себя!

Задача 1.
Составить программу вывода на экран числа, вводимого с клавиатуры. Выводимому числу должно предшествовать сообщение «Вы ввели число».

Задача 2.
Даны целые натуральные числа a и b. Найти все целые числа в диапазоне от a до b, кратные c.

Задача 3.
Дано натуральное число n.
Найти сумму всех его делителей.
Найти сумму всех его четных делителей.
Определить количество всех его делителей.
Определить количество всех его четных делителей.

Задача 4.
Известны оценки каждого студента из группы по двум экзаменам. Определить количество студентов группы, получивших на экзамене тройку.

Задача 5.
Даны целые числа c1, c2, …, c10. Определить среднее арифметическое четных из них.

Задача 6.
Даны натуральное число n и вещественные числа c1, c2, …, cn. Найти:
максимальное из вещественных чисел;
минимальное из вещественных чисел;

Задача 7.
Даны площади нескольких квадратов. Найти длину диагонали самого большого из них.

Задача 8.
Даны вещественные числа a1, a2, …, a10. Верно ли:
что их сумма превышает 100?
что их произведение превышает 10000?

Задача 9.
Даны целые числа a1, a2, …, a10. Выяснить:
а) верно ли, что сумма тех из них, которые больше 30, превышает 142;
б) верно ли, что сумма тех из них, которые меньше 40, есть нечетное число.
